### üßæ Commands

| Command     |Description            |Permissions                  |
| ----------- |---------------------  |------------                 |
| `/sethome`  | Sets a home at your current position                |ghostyhome.sethome |
| `/home`     | Teleport to your home or opens the home GUI         |ghostyhome.home    | 
| `/delhome`  | Deletes one of your homes                           |ghostyhome.delhome |
